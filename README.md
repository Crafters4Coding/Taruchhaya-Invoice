# Taruchhaya Invoice

A beautiful, simple, offline-first invoice and inventory management system designed for ease of use. It stores all data locally in the browser and features a sleek Dashboard, Order Management, Saved Bills, Payment History, and Customer/Product tracking.

## Features

- **Dashboard:** At-a-glance business overview, cash flow charts, and receivables tracking.
- **Order Management:** Select customers, add products, adjust quantities and prices dynamically.
- **Shareable Bills:** Automatically generate shareable, aesthetic invoices (as images) when orders are placed.
- **Offline First:** Data is saved to your browser's local storage—no database setup required.
- **Authentication:** Simple built-in access control.

## Getting Started

1. Clone the repository.
2. Open `index.html` (or `login.html`) directly in your web browser, or serve it via a local development server (e.g., `npx serve .` or `python -m http.server`).
